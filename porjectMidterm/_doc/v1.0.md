# v1.0 — 專業化升級

## 日期
2026-05-21

## 概述
針對三個核心檔案進行專業化升級，加入存檔機制、DataLoader 批次訓練、外部資料讀取、Dropout 防過擬合、Temperature 採樣。

## 變更

### 1. 存檔與讀檔機制（Save/Load Checkpoints）
- `main.py` 訓練後使用 `torch.save()` 儲存 `model_checkpoint.pt`
- 新增 `load_model()` 函數，支援從存檔讀取權重
- 避免每次執行都重新訓練

### 2. DataLoader 批次訓練（Batching）
- `text_processor.py` 新增 `TextDataset` 類別（繼承 `torch.utils.data.Dataset`）
- 新增 `create_dataloader()` 快捷函數
- `main.py` 改用 DataLoader 逐批次訓練（batch_size=4）

### 3. 外部 .txt 檔案讀取（External Dataset）
- `text_processor.py` 新增 `load_external_text(filepath)` 函數
- 嘗試讀取 `dataset.txt`，找不到則使用備用故事（Fallback）
- 處理空檔案與空白檔案的邊界情況

### 4. Dropout 防止過度擬合
- `model.py` LSTM 改為 `num_layers=2`（Dropout 需要至少 2 層）
- 新增 `nn.Dropout(0.2)` 層
- LSTM 內部設定 `dropout=0.2`

### 5. Temperature 採樣機制
- `main.py` 的 `generate_text()` 加入 `temperature` 參數（預設 0.8）
- softmax 前將 logits 除以 temperature
- 展示三種溫度效果：0.5（保守）、0.8（預設）、1.2（創意）

## 新增超參數
| 參數 | 預設值 | 說明 |
|------|--------|------|
| `num_layers` | 2 | LSTM 層數 |
| `dropout` | 0.2 | 隨機遺忘比例 |
| `batch_size` | 4 | DataLoader 批次大小 |
| `temperature` | 0.8 | 文字生成溫度 |

## 測試更新
- 總計：72 項測試（+16）
- 新增測試類別：
  - `TestLoadExternalText`（5 項）
  - `TestTextDataset`（3 項）
  - `TestDataLoader`（4 項）
  - `TestSaveLoad`（2 項）
  - `TestGenerateText` 擴增（2 項）

- 全部通過

```bash
# 一鍵測試

```
