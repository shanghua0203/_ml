# v0.1 — 初始單檔 LSTM 語言模型

## 日期
2026-05-21

## 概述
建立單一 Python 檔案的 LSTM 中文語言模型，包含完整的文字預處理、模型架構、訓練迴圈與文字生成。

## 檔案
- `language_model.py` — 唯一檔案，包含所有功能

## 功能
- 內建 50 字中文小故事作為訓練資料
- 文字預處理：建立字典、文字轉數字、準備訓練樣本
- 模型架構：Embedding + LSTM + Linear（無 Transformer/Attention）
- 訓練迴圈：150 epochs，Adam 優化器，lr=0.01
- 文字生成：multinomial 抽樣，支援不同開頭字

## 超參數
| 參數 | 值 |
|------|-----|
| embed_size | 32 |
| hidden_size | 64 |
| sequence_length | 8 |
| learning_rate | 0.01 |
| total_epochs | 150 |

## 執行方式
```bash
python3 language_model.py
```

## 備註
- 所有程式碼在同一檔案，可直接從頭執行到尾
- 註解使用超級白話風格，適合初學者理解
