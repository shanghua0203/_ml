# v0.2 — 模組化分家

## 日期
2026-05-21

## 概述
將 v0.1 的單一檔案拆解為三個獨立模組，提升程式碼可讀性與維護性。

## 變更
- 移除 `language_model.py`（保留作為參考）
- 新增 `text_processor.py` — 資料處理模組
- 新增 `model.py` — 模型架構模組
- 新增 `main.py` — 主程式模組
- 新增 `README.md` — 專案說明文件

## 檔案職責
| 檔案 | 職責 |
|------|------|
| `text_processor.py` | `build_vocab()`、`text_to_ids()`、`prepare_training_data()` |
| `model.py` | `MyLanguageModel` 類別（Embedding + LSTM + Linear） |
| `main.py` | 匯入前兩者，執行訓練迴圈與文字生成展示 |

## 執行方式
```bash
python3 main.py
```

## 備註
- 保持白話註解風格
- 模型架構不變，僅重組檔案結構
